# Scripts Yarn

- `yarn start` - Serveur de développement
- `yarn build` - Build de production
- `yarn preview` - Prévisualise le build de production
- `yarn lint` - Vérification ESLint
- `yarn lint:fix` - Correction automatique ESLint
- `yarn format` - Formatage Prettier
- `yarn format:check` - Vérification formatage
- `yarn cleanup` - Supprime le dossier `dist` et `node_modules`
